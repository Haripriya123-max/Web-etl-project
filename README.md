# Web Data Extraction and ETL Pipeline

## Project Overview
This project demonstrates a simple ETL (Extract, Transform, Load) pipeline built using Python.  
The pipeline extracts structured data from a public website, cleans and validates the data, and loads it into a database for analysis.

## Tech Stack
- Python
- BeautifulSoup
- Pandas
- SQLite
- SQL
- Git & GitHub
